2D world: Which cities are on a similar latitude?

This is a slightly interactive reproduction of a static graphic by Lisa Rost that she had published on Twitter.
For data source, project documentation and details ideation, take a look at this github repo.

Each line represents the biggest city for each country in case it has more than 2 million inhabitants.
The length of the line corresponds to the city population. Mouse over the lines to learn about population size.
